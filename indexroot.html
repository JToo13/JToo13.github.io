Smartfarm project

This system aims at regulating the soil moisture levels in farms. I designed a program that will automatically detect a drop or surge in soil moisture level and activate a pump to irrigate the farm otherwise the pump will be turned off respectively.

RFID based smart attendance system

Radio Frequency Identification, commonly referred to in short terms as RFID, is the wireless non-contact use of radio frequency to transfer data

How to MQTT series

In the project we used the HIVEMQ broker and installed an MQTT client from the playstore which we used to set up the software side of the project; by subscribing a topic,then publishing the topic and finally writing a message to accompany the published topic. In our case, our message was either LEDON(to light the LED) or LEDOFF(to switch off the LED).

Access Point and Station Mode

In this project we configured the ESP8266 to operate in two different modes. These modes determine what state the nodemcu will be in during project implementation. In Access point mode configuration, the ESP8266 sets up its own Wi-Fi network and acts as a hub(router) for one or more stations while in Station mode the nodemcu connects to an already set up wireless network.

RFID Smart door lock system

In this project we made use nodemcu and the blynk app.The blynk application allowed us to remotely monitor the door lock from anywhere through my smartphone. There was also an LED which would light up when the user has been granted access and stays off if the user has been restricted.

Smart Security system

A smart security system will monitor and access control and alarm systems. In this project whenever motion was detected, the message was sent to our telegram bot hence one can be able to monitor their homes.
